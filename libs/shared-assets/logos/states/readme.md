
# Generic Assets

This folder contains branded logo images for specific ADAPT instances.

## Usage

Place logo files with standardized names (e.g., `state-hero-logo.svg`, `state-nav-logo.png`) within a nested directory named after the state's abbreviation (e.g., `ar` for Arkansas, `hi` for Hawaii). Allowed file types are SVG and PNG.  These logos will be used when the ADAPT instance is configured for that particular state, allowing for customized branding that reflects the identity of each state-specific deployment.
